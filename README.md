Este é um projeto de portal de notícias moderno, focado em tecnologia e inovação, desenvolvido para praticar a estruturação de layouts complexos e responsivos. O site apresenta categorias como Inteligência Artificial, Robótica e Realidade Virtual, utilizando uma interface limpa e intuitiva para o usuário.
Principais Tecnologias e Conceitos Aplicados:
HTML5 Semântico: Organização clara de seções, cabeçalhos e artigos para melhor acessibilidade e SEO.
CSS3 & Flexbox: Uso intensivo de display: flex para alinhar os cards de notícias e o menu de navegação de forma eficiente.
Layout Grid/Flex: Organização das notícias em diferentes tamanhos (destaque principal vs. notícias secundárias), aplicando conceitos de eixo principal e cruzado.
Justify-Content: Distribuição equilibrada dos elementos no layout para garantir um espaçamento profissional entre as notícias.
